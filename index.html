Fast Food (click me)

The admin panel of the fast food site is made using React js, in it you can see products, orders, branches, customers, about employees, delivery costs, reports and locations of branches.

We did this project as a team and I made a layout, a delivery amount page, a report page, a page of branches locations on the map, partly a page of branches and clients

When you enter the site, the login page will appear, you can enter the system by entering the email address fastfood@gmail.com and the password: 123

I used typescript in react on this site. I also left tailwind, antd, mui, yandex maps, redux toolkit. I made the layout part using the layout component of antd, including the delivery price, map, and report pages in the menu. I used tailwind and css file to style the layout.

When making the delivery price, I saved the data from the mocky dev api in the router, using the useEffect hook to get the api and using the dispatch redux toolkit, I stored it in the reducers opened in slices in the store folder. When editing and adding, the drawer opens and you can edit existing data or add new data. Using the form component of antd in the drawer, after filling the form, the values ​​are saved and saved through the api router. All the information is displayed in the table. customers and affiliates work the same way

UseEffect was used in the same way on the map page and the map was generated by the yandex maps library. Through the branches coming from the API, markers are issued based on each branch location. When the markers are clicked, the branch operator and phone number will be displayed

The report page consists of 2 parts. presented in table and chart form. The table calculates the total number of orders in each branch and the income received via payme, cash, and terminal. In the chart part, it shows the income on the day orders were placed for each branch, in which I used the Line component of antd. Here, the information coming from the api made in mocky dev is stored through UseState. Data can be updated by refreshing in the upper part. and the second chart contains statistics made by emotion css, which also shows income by branches and the selected date

Fast Food mini (click me)

In this fast food website, people can order food, select the desired food categories and number of servings, and when placing an order they will need to enter their data so that they can be delivered directly to their home.

this site uses typescript in react vite. I also used tailwind and antd for styling. Redux toolkit is also used, which stores the data coming from the api. A little animation was used via the "react-awesome-reveal" library. React and Antd icons are used. You can order fast food on this site, select the products belonging to different categories and click the add button on them, a modal will appear and information about this product will be taken from the API.

If you specify the number of products here and click the add button, this product will appear in the basket, modal closes. If this product is selected again, the number of products in the basket will be visible and can be changed here. By pressing the button to order products in the basket and entering the delivery address, phone number and name, it will be possible to order the selected products in the basket.

Companies and Jobs (click me)

This site is made using typescript in react. The styles are simplified in tailwind and components of antd are used. The admin panel of this site contains companies and jobs. the Redux toolkit.
